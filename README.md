Zerodha Clone – Full Stack Project:
A full-stack web application inspired by Zerodha, built to understand how real-world trading platforms are developed.
This project was created as a learning and practice project to gain hands-on experience with frontend, backend, database, and deployment concepts.

🛠 Tools & Requirements:
VS Code – Code editor
Google Chrome – For testing and developer tools
Node.js – To build backend and run React app
Git & GitHub – Version control and code hosting

🎨 Frontend Technologies:
Used for building the landing pages and dashboard UI:
HTML
CSS
JavaScript
React.js
Bootstrap
Material UI
Icons libraries:
🔹 React is used to create a Single Page Application (SPA) for better performance and user experience.

⚙️ Backend Technologies:
Node.js
Express.js
🔹 Express.js is used as a backend framework to:
Handle API requests
Manage authentication
Connect frontend with the database

🗄️ Database Configuration:
MongoDB
🔹 Used to store user data and application-related information.

🧪 Testing Framework
Jest
🔹 Jest is used for testing React components and ensuring the application works correctly without manual browser testing.

🚀 Deployment Platform
Amazon AWS (Paid service)

🔧 GitHub Setup
Repository structured with separate folders for:
frontend
backend
dashboard
Version control handled using Git
Code pushed and managed via GitHub

🎯 Purpose of the Project
To understand full-stack web development
To learn how frontend, backend, and database work together
To gain practical experience with real-world application structure
To improve Git and GitHub workflow knowledge

⚠️ Disclaimer

This project is not an official Zerodha application.
It is only a learning project inspired by Zerodha for educational purposes.
